# flo-website
flo Financial Intelligence — official website. Single-page site built in HTML/CSS/JS with Beehiiv RSS integration, Cal.com booking, a live neural-network hero animation, and a scripted FAQ assistant widget (guided prompts + keyword matching, no backend).

## Environment variables (Netlify → Site settings → Environment)
- `BEEHIIV_API_KEY` — Beehiiv API key for the newsletter signup (`netlify/functions/subscribe.js`).
